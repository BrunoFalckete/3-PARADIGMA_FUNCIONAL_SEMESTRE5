Programação Reativa com RxJS

A programação reativa lida de maneira eficiente com eventos e dados assíncronos. No exemplo abaixo, utilizamos o RxJS para criar um Observable a partir de eventos de clique, demonstrando como a programação reativa pode ser aplicada para reagir a eventos de forma declarativa.

Exemplo de Código

            
                document.addEventListener("DOMContentLoaded", () => {
                    const button = document.getElementById("ButtonPR");
                    const output = document.getElementById("output");

                    // Verificando se o botão foi encontrado
                    if (button) {
                        // Adicionando um ouvinte de evento de clique ao botão
                        button.addEventListener("click", () => {
                            output.textContent = "Clique detectado!";
                        });
                    } else {
                        console.error(
                            "Botão não encontrado. Certifique-se de que o ID do botão está correto."
                        );
                    }
                });
