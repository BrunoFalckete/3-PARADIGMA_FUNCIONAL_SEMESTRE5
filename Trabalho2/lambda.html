Lambda

Lambda (Arrow Functions) com Entrada de Dados

Expressões lambda são uma forma concisa de escrever funções em JavaScript. Aqui, usamos uma expressão lambda para definir uma função de soma com entrada de dados.

Exemplo de Código

        
        document.addEventListener("DOMContentLoaded", () => {
            // Definindo uma função de soma usando expressão lambda
            const soma = (a, b) => a + b;

            // Ação do botão no frontend
            const button = document.getElementById("myButton");
            const inputA = document.getElementById("inputA");
            const inputB = document.getElementById("inputB");
            const output = document.getElementById("output");

            button.addEventListener("click", () => {
                // Obtendo valores dos campos de entrada
                const valorA = parseInt(inputA.value);
                const valorB = parseInt(inputB.value);

                // Chamada da expressão lambda para realizar a soma
                const resultadoSoma = soma(valorA, valorB);

                // Exibição do resultado
                output.textContent = `Resultado da soma: ${resultadoSoma}`;
            });
        });
